What is Ebotic?

Our company, Ebotic, is developing a drone that performs tasks autonomously indoors and outdoors via voice commands.
Imagine a phone or a computer, that flies. You will be able to run apps and games on it.

The Problem

  • Businesses that want to sell a drone application need to build it from the ground up (hardware and software).
  • Developers need a simple, safe, elegant platform and toolkit (claw, pico projector, etc.) to build applications without worrying about all the hard stuff (such as obstacle avoidance).

System Overview

Fully Integrated Robotic System

  • Sensors for safety and intelligence
  • Hardware for custom tasking
  • Frame that elegantly seams together all hardware and sensors
  • Software for AI and Path Planning
  • API that integrates our custom sensors, hardware, and software
  • Apps that use the API to create new functionality

What are we aiming for?

  • Our drone will become a part of your everyday life.
  • It will be compatible with a pico-projector, speakers, and much more.
  • Flying gracefully and autonomously in your house, doing chores to make your life easier.

What is our product?

Flying Robot

The flying robot is the hardware platform for this amazing robot partner. It contains sensors like a camera, radar, microphone, GPS, and actuators, like lasers, and possibly pico projectors, a robotic claw, and speakers. All of this will be the base on top of which all the apps will run.

Apps

Independent developers can create their own applications that can use sensors and actuators to solve specific problems, such as doing laundry, walking the dog, picking up food at the counter, and taking selfies/dronies. We will internally develop some apps like home security and package delivery.

Obstacle Avoidance System

Developers will not want to reinvent the wheel; that's why we will provide easy-to-use APIs for obstacle avoidance, indoor mappings and self-localization. That way you can make a new app in a matter of minutes.
