What is Ebotic?

Our company, Ebotic, is developing a ubiquitous technology platform called the Personal Flying Assistant that performs tasks autonomously indoor and outdoor via app and voice command.

The Problem

  • Businesses that want to sell a drone application need to build it from the ground up (hardware and software).
  • Developers need a simple, safe, elegant platform and toolkit (claw, pico projector, etc.) to build applications without worrying about all the hard stuff (obstacle avoidance for example).

System Overview

Fully Integrated Robotic System

  • Sensors for safety and intelligence
  • Hardware for custom tasking
  • Frame that elegantly seamed together all hardware and sensors
  • Software for AI and Path Planning
  • API that integrates our custom sensors, hardware, and software
  • Apps that use the API to create new functions

What are we claiming?

  • The Personal Flying Assistant (PFA) will become a part of your everyday life
  • Compatible with a pico-projector, speakers, and much more
  • Flying gracefully and autonomously in your house performing tasks to make your life easier

What are our products?

Flying Robot

The flying robot is the hardware platform for this amazing assistant. It contains sensors like a camera, radar, microphone, GPS, and actuators like lasers, pico projectors, a robotic claw, and speakers. All of this will be the base on top of which all the apps will run.

Apps

Independent developers can create their own applications that can use sensors and actuators to solve specific problems like doing laundry, walking the dog, picking up food at the counter, and taking selfies/dronies. We will internally develop some apps like home security and package delivery.

Obstacle Avoidance System

Developers will not want to reinvent the wheel; that's why we will provide easy-to-use APIs for obstacle avoidance, indoor mappings and self-localization. That way you can make a new app in a matter of minutes.
